# Лабораторна робота №2: Засоби підготовки та аналізу даних

## Опис завдання
Ця лабораторна робота присвячена дослідженню та візуалізації даних за допомогою Python і складається з двох частин:
1. **Частина 1:** Отримання, очищення та переіндексація даних індексу VHI (Vegetation Health Index) з серверів NOAA. Робота з вибірками та пошук екстремумів за допомогою бібліотеки `pandas`.
2. **Частина 2:** Аналіз великого датасету електроспоживання (Household Power Consumption). Профілювання та порівняння швидкодії виконання складних вибірок між структурами даних `pandas` та масивами `numpy` за допомогою модуля `timeit`. Виконання статистичних перетворень та One Hot Encoding.

## Вимоги до системи
* **ОС:** Windows / macOS / Linux
* **Інтерпретатор:** Python 3.10+ 
* **IDE:** VS Code або Jupyter Notebook
