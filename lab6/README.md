## Опис проєкту
Цей проєкт реалізує модульну програму на C++ з використанням бібліотеки комп'ютерного зору **OpenCV**. Програма захоплює відеопотік з веб-камери в реальному часі, відображає його та дозволяє користувачу інтерактивно застосовувати різні фільтри та ефекти обробки зображень. 

Проєкт побудовано з дотриманням принципів об'єктно-орієнтованого програмування (ООП). Логіка розділена на окремі класи для роботи з камерою, обробки натискань клавіш та накладання графічних ефектів.

## Функціональні можливості
* **Захоплення відео:** Безперервне читання кадрів з підключеної камери (`CameraProvider`).
* **Модульна обробка кадрів (`FrameProcessor`):**
  * `Normal (Оригінал)` — базове відображення без змін.
  * `Inversion` — інверсія кольорів (негатив).
  * `Gaussian Blur` — розмиття зображення.
  * `Canny Edge Detection` — виділення контурів об'єктів.
  * `Glitch Effect` — ефект зміщення кольорових каналів (RGB-спліт).
* **Інтерактивність:** 
  * Керування режимами за допомогою клавіатури (`KeyProcessor`).
  * Трекбар (слайдер) для динамічної зміни параметрів ефектів (наприклад, яскравості або інтенсивності розмиття).
* **Аналітика в реальному часі:** Відображення на екрані показника FPS (кадрів на секунду) та активного режиму обробки.

## Вимоги до системи та обладнання
Для успішної компіляції та запуску програми ваша система повинна відповідати наступним вимогам:
* **Операційна система:** Ubuntu 24.04 LTS (або інші сумісні Linux-дистрибутиви).
* **Компільтор:** GCC / G++ (з підтримкою стандарту C++17 або вище).
* **Система збірки:** CMake (версія 3.10+), Make.
* **Бібліотеки:** OpenCV 4.x (модулі core, videoio, highgui, imgproc).
* **Апаратне забезпечення:** Наявність веб-камери (вбудованої або підключеної через USB).

## Інструкція з налаштування та запуску

Проєкт містить зручні bash-скрипти для повної автоматизації процесу розгортання.

**Клонування репозиторію та перехід у теку**
```bash
git clone <посилання_на_ваш_репозиторій>
cd <назва_репозиторію>/lab6
```
**Надання прав на виконання скриптам**
Перед першим запуском необхідно зробити скрипти виконуваними:
```
chmod +x preinstall.sh build.sh run.sh
```
**Встановлення залежностей**
Цей скрипт автоматично встановить CMake, компілятор та необхідні пакети OpenCV:
```
./preinstall.sh
```
**Компіляція проєкту**
Скрипт створить директорію build, згенерує файли збірки через CMake та скомпілює бінарний файл:
```
./build.sh
```
**Запуск програми**
```
./run.sh
```
