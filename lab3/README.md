# Лабораторна робота №3. Візуалізація даних

**Дисципліна:** Збір та попередній аналіз даних (ЗПАД)

## Опис
У цій лабораторній роботі виконано аналіз та візуалізацію багатовимірного набору даних **Automobile Dataset** (з репозиторію UCI Machine Learning). 

Основна мета роботи — практичне застосування бібліотек Python для підготовки даних, побудови інформативних графіків та виявлення прихованих закономірностей і кореляцій між технічними характеристиками автомобілів.

## Виконані завдання:
1. **Data Cleaning:** Очищення даних від невалідних символів (`?`), заповнення пропусків у числових ознаках медіаною, а в категоріальних — модою. Видалення записів з відсутньою цільовою змінною (ціною).
2. **Аналіз залежностей (Scatter Plot):** Побудова графіка залежності ціни від ваги автомобіля з накладанням лінії тренду.
3. **Аналіз розподілу (Histogram):** Побудова гістограми витрат пального (highway-mpg) з розбиттям чітко на 5 діапазонів.
4. **Багатовимірна візуалізація:** Відображення одразу 4-х вимірів (потужність, ціна, тип приводу, об'єм двигуна) на одній площині для пошуку складних взаємозв'язків.
5. **Деталізований розподіл (Boxplot + Swarmplot):** Комбінований аналіз вартості автомобілів залежно від типу кузова.
6. **Кореляційний аналіз (Heatmap):** Побудова теплової карти кореляції Пірсона для ключових числових характеристик.

## Стек технологій
* Python 3
* Pandas, NumPy
* Matplotlib, Seaborn
* Jupyter Notebook

## Запуск
1. Створіть та активуйте віртуальне середовище: python -m venv venv
2. Встановіть необхідні залежності: pip install -r requirements.txt
3. Запустіть Jupyter Notebook: jupyter notebook
4. Відкрийте файл з кодом лабораторної роботи та виконайте комірки по черзі.
