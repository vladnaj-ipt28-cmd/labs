## Опис проєкту
]Метою цієї роботи є обмін результатами дослідження та надання зовнішнім користувачам можливості зручно працювати з отриманими даними. Цей проєкт реалізує інтерактивний вебдодаток за допомогою бібліотеки `Streamlit`.Додаток дозволяє аналізувати та візуалізувати часові ряди вегетаційних індексів (VCI, TCI, VHI) на основі набору даних із лабораторної роботи №2[cite: 48].

### Основний функціонал:
**Панель налаштувань:** * Випадні списки (dropdown) для вибору індексу (VCI, TCI, VHI) та області України.
  * Слайдери для точного налаштування діапазону тижнів та років для аналізу.
  **Три вкладки для аналізу:**
  * **Таблиця даних:** Відображає відфільтровані значення. Містить систему сортування (за зростанням/спаданням) за допомогою чекбоксів.Якщо обрано обидва чекбокси одночасно, програма залишає дані без змін та видає попередження.
    **Графік індексу:** Лінійний графік, що відображає динаміку обраного індексу для конкретної області у заданому часовому інтервалі.
    **Порівняння областей:** Стовпчаста діаграма для порівняння значень обраного індексу обраної області з усіма іншими областями.
  **Скидання фільтрів:** Кнопка "Скинути фільтри", яка миттєво повертає всі інтерактивні елементи до їх початкових значень.
  **UI/UX:** Інтерактивні елементи керування винесені в окрему бокову колонку зліва, а графіки та таблиця займають основну широку частину екрана справа.
