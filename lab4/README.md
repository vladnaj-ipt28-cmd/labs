## Опис проєкту
Цей проєкт реалізує програму з графічним інтерфейсом користувача (GUI) для генерації, аналізу та фільтрації гармонійних сигналів. Програма дозволяє дослідити вплив гаусівського шуму на ідеальну синусоїду та ефективність роботи цифрових фільтрів.

### Основні можливості:
* **Генерація сигналу:** Побудова графіка функції виду $y(t) = A\sin(\omega t + \phi)$.
* **Управління шумом:** Додавання шуму із заданим середнім значенням та дисперсією. Базовий масив шуму генерується одноразово, тому при зміні параметрів гармоніки сам візерунок шуму залишається незмінним.
* **Фільтрація:** Використовується низькочастотний фільтр Баттерворта (`scipy.signal.butter` та `filtfilt`). Користувач може в реальному часі змінювати частоту зрізу фільтра, щоб зробити відфільтровану криву максимально наближеною до ідеальної гармоніки.
* **Інтерактивність:** Наявні слайдери для всіх параметрів, чекбокс для швидкого вмикання/вимикання шуму та кнопка скидання налаштувань.

## Інструкція для користувача
1.  **Встановлення залежностей:** Переконайтеся, що у вас встановлено Python. Відкрийте термінал у папці з проєктом та виконайте команду:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Запуск програми:**
    Запустіть головний скрипт:
    ```bash
    python lab4.py
    ```
3.  **Використання інтерфейсу:**
    * **Слайдери "Амплітуда", "Частота", "Фаза":** Змінюють параметри червоної кривої (чистої гармоніки).
    * **Слайдери "Шум (Середнє)", "Шум (Дисперсія)":** Керують характеристиками помаранчевого зашумленого сигналу.
    * **Слайдер "Зріз фільтра":** Керує частотою відсікання для фільтра Баттерворта. Зменшуйте значення, щоб згладити синю пунктирну лінію, але слідкуйте, щоб не "зрізати" саму гармоніку.
    * **Чекбокс "Шум ON/OFF":** Вимикає накладання шуму, дозволяючи подивитись на роботу програми в ідеальних умовах.
    * **Кнопка "Reset":** Повертає всі повзунки до початкових значень.
